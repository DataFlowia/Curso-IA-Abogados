📑 Conceptos Fundamentales

Definición de IA

Sistemas que imitan inteligencia humana para tareas específicas como analizar patrones o generar texto.

IA Estrecha vs. General

IA limitada a funciones puntuales (ej. revisión de contratos) versus IA universal (hipotética).

Analogía Básica

La IA es como un asistente que aprende de datos masivos; no es mágica ni infalible.

📚 Glosario Esencial & Terminología Técnica

Algoritmo

Serie de instrucciones paso a paso para resolver un problema.

Agentes

Sistemas autónomos que pueden realizar tareas y tomar decisiones limitadas.

AGI

Inteligencia Artificial General: IA con capacidad humana para cualquier tarea intelectual.

API

Interfaz que permite que dos aplicaciones se comuniquen entre sí.

Hosting / Alojamiento

Espacio en un servidor donde "viven" los archivos de una aplicación.

Dominio

La dirección exclusiva de su sitio web (ej: www.estudiojuridico.com).

Base de Datos

Almacén estructurado de información digital.

Web Estática

Sitio web con contenido fijo que se entrega al usuario tal cual.

Plataformas

Entornos digitales donde se ejecutan aplicaciones (ej: OpenAI, Claude).

🛡️ Ética y Confidencialidad

El marco de seguridad estricto para manejar datos legales sensibles sin comprometer el secreto profesional.
